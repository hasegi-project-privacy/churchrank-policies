이용약관

교회순위 (Korean Church Ranking)
최종 개정 2026-04-29 · 시행 2026-04-29

본 약관은 Hasegi Inc.(이하 "운영자")가 제공하는 모바일 애플리케이션 교회순위(이하 "서비스")의 이용에 관한 운영자와 이용자 간의 권리·의무 및 책임 사항을 규정합니다. 서비스를 이용하는 행위는 본 약관에 동의하는 것으로 간주됩니다.

01목적

본 약관은 이용자가 서비스를 안전하고 합리적으로 이용할 수 있도록 필요한 사항을 정함을 목적으로 합니다.

02용어의 정의

  • 서비스 — 한국 교회 YouTube 채널의 공개 통계를 정리하여 순위 형태로 제공하는 모바일 애플리케이션 "교회순위".
  • 이용자 — 본 약관에 따라 서비스를 설치하고 이용하는 모든 사람.
  • 채널 통계 — YouTube Data API v3를 통해 공개된 구독자수·조회수·구독전환률 등의 정보.
  • 즐겨찾기 — 이용자가 자신의 기기에 등록한 관심 채널 목록.

03약관의 효력 및 변경

본 약관은 앱 내 또는 본 페이지에 게시함으로써 효력이 발생합니다. 운영자는 관련 법령을 준수하는 범위 내에서 약관을 개정할 수 있으며, 개정 시 시행일 7일 전(이용자에게 불리한 변경의 경우 30일 전)에 앱 내 또는 본 페이지를 통해 공지합니다. 이용자가 변경된 약관 시행일 이후에도 서비스를 계속 이용하는 경우 변경에 동의한 것으로 간주합니다.

04서비스의 제공

  1. 한국 교회 YouTube 채널 약 100여 곳의 일별 순위 표시 (구독자수·조회수·구독전환률 기준 정렬).
  2. 전일 대비 순위 변동 및 7일간 구독자 증가량 표시.
  3. 이용자가 직접 추가한 임의 YouTube 채널을 즐겨찾기에 등록·관리하는 기능.
  4. 등록된 채널의 YouTube 페이지로 즉시 이동하는 외부 링크 기능.

서비스 데이터는 매일 한국 표준시(KST) 오전 8시 1회 자동 갱신됩니다.

05서비스의 변경 및 중단

운영자는 서비스 개선, 기술적 사정, 외부 API 정책 변경 등의 사유로 서비스의 일부 또는 전부를 변경하거나 중단할 수 있으며, 이로 인해 이용자에게 발생한 손해에 대해 무상 서비스의 한도 내에서 책임을 지지 않습니다.

06데이터의 정확성에 관한 면책

  • 모든 채널 통계는 YouTube Data API v3로부터 수집한 공개 데이터이며, 운영자가 직접 가공·평가한 값이 아닙니다.
  • 일별 1회 갱신 정책상 실시간 데이터와 차이가 있을 수 있습니다.
  • 채널 운영 종료, 채널명 변경, API 응답 누락 등의 사유로 일시적인 표시 오류가 발생할 수 있습니다.
  • 운영자는 데이터의 완전성·정확성·시의성을 보장하지 않으며, 이로 인한 직간접적 손해에 대해 책임지지 않습니다.
  • 특정 교회·단체의 노출 또는 미노출은 추천·평가·승인이 아닙니다.

07광고

서비스는 Google AdMob을 통한 광고를 포함합니다. 이용자가 시청한 광고의 내용·정확성·적법성에 대한 책임은 광고주 및 AdMob에 있으며, 운영자는 광고로 인해 발생한 거래·분쟁에 관여하지 않습니다.

08이용자의 의무

  • 이용자는 서비스를 합법적인 목적으로만 이용해야 합니다.
  • 다음 행위는 금지됩니다.
    • 서비스의 정상 운영을 방해하는 자동화된 접근(봇, 스크레이퍼 등)
    • 역공학 / 디컴파일 / 무단 복제 / 재배포
    • 제3자의 권리(저작권·상표권·인격권 등) 침해
  • 외부 채널을 즐겨찾기에 추가할 때, 해당 채널의 공개 정보 외에는 어떠한 권리 주장도 하지 않습니다.

09지식재산권

서비스의 디자인·코드·문구·구성에 관한 지식재산권은 운영자에게 귀속되며, YouTube 채널 정보 및 채널명·로고 등은 각 채널 운영자에게 귀속됩니다.

10분쟁의 해결

본 약관은 대한민국 법률에 따라 해석되며, 서비스 이용과 관련하여 운영자와 이용자 사이에 분쟁이 발생한 경우 양 당사자는 성실한 협의를 통해 해결을 시도합니다. 협의가 이루어지지 않을 경우 「민사소송법」상의 관할 법원을 합의관할로 합니다.
